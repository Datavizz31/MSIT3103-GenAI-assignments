# MSIT3103 Generative AI Assignments

## Assignment 1: Fashion-MNIST Analysis

This project analyzes the Fashion-MNIST dataset using traditional machine learning models and implements a basic GAN from scratch.

### What's Included

1. **Traditional Models**: Logistic Regression, SVM, and Decision Tree comparison
2. **Basic GAN**: NumPy-only implementation generating 8×8 fashion images

### Files

- `assignment1_hotstart.ipynb` - Main notebook with all code and analysis
- `README.md` - This file

### Requirements

```
numpy
matplotlib
scikit-learn
```

### How to Run

Open `assignment1_hotstart.ipynb` in Jupyter and run all cells.

### Key Results

- **Best Traditional Model**: Logistic Regression (highest accuracy)
- **GAN**: Successfully generates fashion-like patterns from random noise
- **Comparison**: Traditional models better for classification, GAN useful for data generation
